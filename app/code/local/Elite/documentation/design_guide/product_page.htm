Display Vehicle Applications

To display vehicles [applications] for a product, open app\design\frontend\base\default\template\catalog\product\view.pthml and paste the following snippet where you would like the fits to appear. Warning: you should not use this feature / or just modify this to take appropriate action if you have more then say, 20 fits or so ( ex. you don't want a list of 1000s of fits on the product page ). 
<?php
if( $_product->isUniversal() )
{
    echo 'Universal Product';
}
else if( count( $_product->getFits() ) )
{
    ?>
    <div class="description">
    <h3>Vehicle Fits</h3>
    <?php
    foreach( $_product->getFitModels() as $fit )
    {
        ?>
        <? echo htmlentities( $fit )?>
        <br />
        <?php
    }
    ?>
    </div>
    <?php
}
?>
