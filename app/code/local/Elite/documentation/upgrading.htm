Upgrading the extension

To upgrade, copy the upgrade-vaf.php to your Magento install path (usually web-root) and run it through the browser. When upgrading, you may need to tweak various customizations you have made. Feel free to contact support at any time.
Upgrading to 1.35 The need for the javascript to be pasted into head.phtml has been removed. Navigate to the head.phtml for the front-end & back-end themes, and remove the javascript/CSS lines. The extension includes these automatically after the form now.
